# SmartSwap 🛍️

SmartSwap is a real-time college marketplace web app where students can buy and sell second-hand items.

## 🚀 Features
- User authentication (Firebase)
- Real-time item listing
- Chat system (buyer ↔ seller)
- Notifications with badge
- Wishlist functionality
- Price filter & search
- Responsive UI
- Forgot password

## 🛠️ Tech Stack
HTML, CSS, Bootstrap, JavaScript, Firebase
